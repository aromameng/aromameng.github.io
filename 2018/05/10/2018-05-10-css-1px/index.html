移动端1px问题的解决方案

发表于 2018-05-10 | 分类于 css

产生原因

因为retina屏的分辨率始终是普通屏幕的2倍,1px的边框在devicePixelRatio=2的retina屏下会显示成2px,所以在高清屏下看着1px总是感觉变大了

主要有以下几种解决方式：

使用scale缩放

这是之前淘宝移动端采取的方案
