vue-cli简单安装

发表于 2017-02-15

vue-cli 是vue.js的脚手架，用于自动生成vue.js模板工程的。

安装vue-cli

npm install -g vue-cli                       //全局安装vue-cli
vue init webpack projectName        //生成项目名为projectName的模板，这里的项目名自定义
cd projectName                              
npm install                                      //安装依赖

PS.作为新手还是将eslint的验证去掉，语法太严格了，吃不消

运行 npm run dev

在浏览器打开 http://localhost:8080，就可以看到欢迎页了
